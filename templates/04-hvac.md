## Statuses

### Planning Phase
- **New** — Lead/enquiry just received
- **Quoted by Phone** — Quick phone quote given (no formal quote)
- **Site Visit Booked** — Assessment visit scheduled
- **Site Visit Complete** — Assessed, ready to quote

### Quoting Phase
- **Quoting** — Formal quote being prepared
- **Quote Sent** — Sent to customer
- **Follow-Up** — Chasing customer response
- **Accepted** — Customer accepted quote

### Delivery Phase
- **Scheduled** — Job booked into the calendar
- **In Progress** — Work underway
- **Waiting on Materials** — Parts ordered, waiting for delivery
- **Waiting on Customer** — Need access, decision, or approval
- **Warranty/Recall** — Return visit for warranty work

### Completion Phase
- **Complete** — Work finished
- **Invoice Sent** — Invoice issued
- **Paid** — Payment received
- **Cancelled** — Job cancelled

## Checklist Templates

### Split System Installation (Single Head)
- [ ] Confirm site access and power availability
- [ ] Measure and mark outdoor unit location (level, clear space, drainage)
- [ ] Install mounting bracket for outdoor unit
- [ ] Position and secure outdoor unit
- [ ] Measure pipe run (minimise length, slope for drainage)
- [ ] Cut wall sleeve and fit wall sleeve trim
- [ ] Route copper piping (3/8" liquid, 1/4" suction)
- [ ] Insulate all piping (R32 compliant wrap)
- [ ] Connect copper lines to outdoor unit
- [ ] Install drain pan and drain line
- [ ] Mount indoor head unit bracket
- [ ] Position indoor head unit
- [ ] Connect refrigerant lines to indoor head
- [ ] Connect condensate drain
- [ ] Run electrical supply to outdoor unit (isolator + circuit breaker)
- [ ] Run low-voltage signal cable (indoor head to outdoor unit)
- [ ] Pressure test entire system (nitrogen only — NO AIR)
- [ ] Evacuate system (deep vacuum, min 10 microns)
- [ ] Charge system with correct refrigerant (R32 or R410A per spec)
- [ ] Test cooling and heating cycles
- [ ] Check temperature differential (indoor/outdoor)
- [ ] Measure superheat/subcooling (if available)
- [ ] Commission unit (set thermostats, operating modes)
- [ ] Clean up site and dispose of packaging
- [ ] Provide customer handover (remote control, operation guide, warranty info)
- [ ] Take completion photos
- [ ] Complete ARCTICK compliance if applicable

### Ducted System Installation
- [ ] Site survey: identify roof/ceiling space, ductwork routing
- [ ] Plan ductwork layout (minimise resistance, balanced distribution)
- [ ] Install outdoor unit (level, secure, drainage, clearance)
- [ ] Install indoor unit in ceiling space (secure, condensate drain to sump)
- [ ] Install return air box (plenum)
- [ ] Route supply ductwork (insulated, sealed, min 200mm diameter for 6.8kW)
- [ ] Route return air ductwork (undersize to create pressure)
- [ ] Install ceiling vents/dampers in all zones
- [ ] Connect copper lines (outdoor to indoor unit)
- [ ] Install electrical supply (dedicated circuit, isolator)
- [ ] Install thermostats (main + zone controllers if applicable)
- [ ] Run low-voltage signal cables
- [ ] Pressure test (nitrogen, no air)
- [ ] Evacuate system (deep vacuum)
- [ ] Charge with correct refrigerant
- [ ] Test all air flow, temperature balance
- [ ] Balance dampers for even distribution
- [ ] Commission thermostats and zone damper controllers
- [ ] Ductwork pressure test (500Pa positive, 100Pa negative)
- [ ] Filter installation and fitting
- [ ] Customer handover and operation training
- [ ] Complete photos and compliance documentation

### Annual Service (Any System)
- [ ] Isolate power to outdoor and indoor units
- [ ] Inspect indoor unit filter (clean or replace if clogged)
- [ ] Clean indoor unit fins and coil (gentle brush, no water pressure)
- [ ] Inspect drain pan and drain line (clear blockages, treat with biocide)
- [ ] Check indoor unit mounting and connections
- [ ] Inspect outdoor unit (debris, damage, security)
- [ ] Clean outdoor unit condenser coil (soft brush, low-pressure rinse)
- [ ] Check outdoor unit fan operation
- [ ] Visually inspect copper piping (leaks, damage, insulation condition)
- [ ] Check all electrical connections (torque, no corrosion)
- [ ] Power on and test in heating and cooling modes
- [ ] Measure inlet/outlet temperatures (log readings)
- [ ] Check for refrigerant leaks (electronic detector or soapy water)
- [ ] If leaks found: isolate, repair, pressure test, evacuate, recharge
- [ ] Listen for abnormal compressor noise
- [ ] Check vibration isolation mounts
- [ ] Run system for 15+ minutes in each mode
- [ ] Confirm temperature control stability
- [ ] Check remote control functionality
- [ ] Provide service report to customer
- [ ] Schedule next annual service
- [ ] Issue service sticker/label

## AI Voice Agent (Ava) — HVAC Script Additions

### Key Questions Ava Asks
- "Are you looking to cool, heat, or both?"
- "Is this for residential, commercial, or light commercial use?"
- "Approximately how many rooms or what size area do you need to cool/heat?"
- "Do you have existing ducting, or would you prefer a ductless split system?"
- "Is this for a new installation, replacement, or breakdown repair?"
- "When would you ideally like the work completed?"
- "Is the system currently not working, or is this a planned upgrade?"
- "Do you know the make and model of your current system?" (if replacement/repair)

### Ava Qualifying Logic
- Emergency breakdown (no cooling in summer, heating in winter) → Flag urgent, attempt same-day callback
- New split system installation → Medium-high value lead, schedule site visit
- Ducted system → High-value lead, schedule detailed site survey
- Annual maintenance/service → Recurring revenue, offer service plan
- Warranty/replacement enquiry → Priority, check warranty coverage first
- Commercial/multi-head VRF → High value, escalate to senior technician assessment

## Customisation Points (Per Client)

When onboarding an HVAC client, customise:

1. **Their specific services** — Not all HVAC contractors do refrigeration, VRF, or large commercial
2. **Their pricing** — Update product catalog with their actual supplier costs and sell prices (Reece, Tradelink, Appli Parts, CRC Group, etc.)
3. **Their suppliers** — Preferred refrigerant sources, electrical, copper suppliers
4. **Their service area** — Suburbs/postcodes they cover; callout fees may vary by distance
5. **Their team** — Staff names, ARCTICK licence numbers, vehicle colours
6. **Their business hours** — Standard hours, after-hours availability, emergency response time
7. **Their existing customers** — Import CRM data from previous system
8. **ARCTICK/License compliance** — Update checklist templates to reflect specific certifications held
9. **Ava's greeting** — Business name, primary services (e.g., "split systems and ducted" vs. "full commercial HVAC")
10. **Warranty offerings** — Extend to 2–5 years if offered; update scope of works text
11. **Service plan defaults** — Residential annual vs. commercial quarterly or monthly intervals
12. **Emergency callout policy** — Same-day response, minimum 4hr window, or on-call roster

## Australian Standards & Compliance

### Key References
- **AS/NZS 5149** — Safety requirements for refrigerating systems and heat pumps
- **AS 3823.1** — Warm air heating systems (ducted)
- **ARCTICK Licensing** — Mandatory for handling refrigerants; verify technician accreditation per state
- **Energy Rating Label** — MEPS (Minimum Energy Performance Standards) for residential split systems
- **AS/NZS 1668.2** — Ventilation and air conditioning systems (commercial)

### Technician Requirements
- ARCTICK Category 1 licence (minimum) for any refrigerant handling
- State-specific electrical licensing if customer-owned electrical work
- Plumbing licence if installing condensate drainage ties
- Yellow Card or equivalent respiratory protection if working in dusty environments

### Checklist Compliance Notes
- All pressure testing must use **nitrogen ONLY** — no air or oxygen (explosion risk)
- Evacuation must reach **10 microns or better** (micron gauge required)
- Final commissioning must document **superheat/subcooling** (if accessible) or temperature differential
- All systems must be tagged/labelled with **date of last service** and **next service due date**
